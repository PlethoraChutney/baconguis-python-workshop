Homework/exercises

Since it's a new restaurant, we can assume that each customer will pick randomly from the menu (nobody know's what's good, and your staff will say everything is equally good).

Here's a bit of code to get a random number generator function, randint(). It's okay if you don't understand the first line yet: 


from random import randint

# this returns a random integer from 0 to 5, inclusive
randint(0, 5)
