Python Fundamentals

Python Fundamentals

  • Python is an interpreted language
  • Python is dynamically typed
  • Python makes extensive use of modules
  • Python is designed to be easy to read and write

What is an interpreted language?

The written code can be executed directly by the python interpreter, without a compilation step

Interpreted languages are generally faster to write but slower to run.

Python Modules

Python comes pre-packaged with tons of modules to do things like generate random numbers, work with new or more complex data structures, interact with the operating system, etc.

In addition, there is an extremely robust python community which contributes tons of free modules for download --- for instance, Appia!

Practice 1.2: Variables

You're on a vacation in Washington and want to buy a souvenir. You see a mug that will remind you of the trip and decide to buy it. It costs $11.95 . Store this value in a variable called mug_price

Sales tax in Washington is 6.5%. Replace the value in mug_price with the price after tax. Use the variable, don't calculate from scratch!

Uh oh! This shop expects you to pay a service charge to the person working the register. The service charce is $3.50. Print the total price so you know how much to pay for the mug, including tax and service charge.
